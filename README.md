## HOW-TO-USE
1. Set `PATH` to `ndk-build`
2. Clone this repository
```
git clone https://github.com/huntto/Build-FAAD2-for-Android.git --recursive
```
3. Run build shell
```
cd Build-FAAD2-for-Android
./android_build.sh
```
